CentOS-KVM配置Bridge-Vlan

发表于 2019-06-04 更新于 2019-10-06 分类于 VLAN ， KVM 阅读次数：
本文字数： 0 阅读时长 ≈ 1 分钟

基本信息

  • 在使用Vlan之前，需要加载8021q
    modprobe 8021q
  • 如果系统已经加载可以不用执行以上步骤
    lsmod | grep 8021q
  • 使用epel-release安装vconfig
    sudo yum install vconfig -y
  • 停用NetworkManager服务
    sudo systemctl disable NetworkManager && sudo systemctl stop NetworkManager


操作步骤

  1. 分别创建2个网卡eth0.1,eth0.2和2个网桥br0.1,br0.2

  2. 分别配置网卡eth0.1和eth0.2

  3. 分别配置网桥br0.1,br0.2

  4. 分别将创建的网卡和网桥启动

  5. 查看网桥信息
    brctl show

  6. 创建3个虚拟机，分别连接到各自的网桥

  7. 分别启动3个vm

  8. 分别设置vm的ip地址

  • cirros:192.168.1.10
  • cirros-1:192.168.1.11
  • cirros-2: 192.168.2.10

  1. 查看网桥信息，可以看到vm的网卡的挂载情况
    brctl show
  2. 测试网络连通情况
    cirros<->cirros-1
    ping 192.168.1.12
    cirros<-x-x->cirros-2
    ping 192.168.2.10
    cirros-1<-x-x->cirros-2
